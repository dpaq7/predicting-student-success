# Predicting Student Success

Welcome to the Predicting Student Success project documentation!

## 📌 Overview

This project aims to predict student success using machine learning techniques. The project follows modern data science best practices with a focus on reproducibility, code quality, and maintainability.

## 🎯 Project Goals

- Develop accurate models for predicting student academic success
- Identify key factors that contribute to student performance
- Create a reproducible and maintainable data science pipeline
- Provide actionable insights for educational institutions

## 🚀 Quick Start

To get started with this project:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/predicting-student-success.git
   cd predicting-student-success
   ```

2. **Set up the environment**:
   ```bash
   make init_env
   source .venv/bin/activate
   ```

3. **Install data science libraries**:
   ```bash
   make install_data_libs
   ```

4. **Run the example notebook**:
   ```bash
   make notebook
   ```

## 🛠️ Technology Stack

- **Python 3.10+**: Core programming language
- **UV**: Fast Python package and environment manager
- **Pandas & NumPy**: Data manipulation and analysis
- **Scikit-learn**: Machine learning algorithms
- **Jupyter**: Interactive development
- **Pytest**: Testing framework
- **MkDocs**: Documentation generation
- **Ruff**: Fast Python linter and formatter
- **Pre-commit**: Git hooks for code quality
